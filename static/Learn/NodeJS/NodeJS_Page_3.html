What are Modules

Modules in Node.js are reusable blocks of code that encapsulate functionality. They allow you to organize your code into separate files and folders, making it easier to maintain and reuse. Modules can be imported into other files using the `require` function. 


// Example of a module exporting a function
// math.js
module.exports = {
   add: function(a, b) {
       return a + b;
   },
   subtract: function(a, b) {
       return a - b;
   }
};

// Example of importing and using the module
// app.js
const math = require('./math.js');

console.log(math.add(5, 3)); // Output: 8
console.log(math.subtract(10, 4)); // Output: 6

Explanation :
In this example, we create a module named `math.js` that exports two functions: `add` and `subtract`. These functions perform addition and subtraction, respectively. In the `app.js` file, we import the `math` module using the `require` function and then use its exported functions to perform mathematical operations.


**Importing Pre-built Modules**

In Node.js, you can easily import pre-built modules provided by the Node.js core or third-party libraries using the `require` function. These modules extend the functionality of your application and allow you to perform various tasks without reinventing the wheel. 


// Example of importing a core module (fs module for file system operations)
const fs = require('fs');

// Example of importing a third-party module (express.js for web server)
const express = require('express');

Explanation :
In this example, we import two types of modules:
1. Core modules: These are modules provided by Node.js core, such as `fs` module for file system operations.
2. Third-party modules: These are modules developed by third-party developers and available through the npm (Node Package Manager) registry. Here, we import the `express` module, which is a popular web framework for building web servers and web applications.

Using Modules

Once you have imported pre-built modules into your Node.js application, you can use them to leverage their functionality. Here's an example of how to use the express module for creating a simple web server: 


// Run this in Terminal
npm install express
      

// Importing the express module
const express = require('express');

// Creating an instance of the express application
const app = express();

// Handling GET requests to the root URL
app.get('/', (req, res) => {
   res.send('Hello, World!');
});

// Starting the server on port 3000
app.listen(3000, () => {
   console.log('Server is running on port 3000');
});

Express.js

Express.js is a web application framework for Node.js, designed for building web applications and APIs. Below are some examples demonstrating various features of Express.js: 


// Example 1: Basic server setup
const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
   res.send('Hello, World!');
});

app.listen(port, () => {
   console.log(`Server is running on port ${port}`);
});

Explanation :

  • const express = require('express');: Imports the express module.
  • const app = express();: Creates an instance of the express application.
  • const port = 3000;: Defines the port number on which the server will listen.
  • app.get('/', (req, res) => { ... });: Defines a route handler for the root URL ('/') that sends a simple text response ('Hello, World!') when a GET request is received.
  • app.listen(port, () => { ... });: Starts the server on the specified port and logs a message to the console once the server is running.


// Serving static files
app.use(express.static('public'));

Explanation :

  • app.use(express.static('public'));: Configures Express to serve static files from the public directory. This allows files such as HTML, CSS, and client-side JavaScript to be served to the client.

Multer

Multer is a middleware for handling multipart/form-data, primarily used for uploading files. Below is an example demonstrating how to use Multer to upload files in an Express.js application: 


// Run this in Terminal
npm install multer
      

// Example: File Upload with Multer
const express = require('express');
const multer = require('multer');
const app = express();
const port = 3000;

// Configure Multer
const storage = multer.diskStorage({
   destination: function (req, file, cb) {
       cb(null, 'uploads/');
   },
   filename: function (req, file, cb) {
       cb(null, file.originalname);
   }
});

const upload = multer({ storage: storage });

// Handle file upload
app.post('/upload', upload.single('file'), (req, res) => {
   res.send('File uploaded successfully');
});

app.listen(port, () => {
   console.log(`Server is running on port ${port}`);
});

Explanation :

  • const multer = require('multer');: Imports the Multer middleware.
  • const storage = multer.diskStorage({ ... });: Configures the storage engine for Multer. In this example, files are stored on disk with a destination directory specified as uploads/ and the original filename.
  • const upload = multer({ storage: storage });: Creates a Multer instance with the specified storage configuration.
  • app.post('/upload', upload.single('file'), (req, res) => { ... });: Defines a route handler for handling file uploads. In this example, it expects a single file upload with the field name 'file'.

Connection to HTML

To connect your Express.js application with HTML, you can use templating engines like EJS (Embedded JavaScript) or Pug (formerly known as Jade). Here's how you can render HTML files and pass data to them using EJS: 


// Example: Connection to HTML with EJS
const express = require('express');
const app = express();
const port = 3000;

// Set the view engine to EJS
app.set('view engine', 'ejs');

// Define a route to render HTML
app.get('/', (req, res) => {
   const data = {
       title: 'Hello, World!',
       message: 'Welcome to my Express.js application!'
   };
   res.render('index', { data });
});

app.listen(port, () => {
   console.log(`Server is running on port ${port}`);
});
       

<html>
  <head>
      <title><%= data.title %></title>
  </head>
  <body>
      <h1><%= data.title %></h1>
      <p><%= data.message %></p>
  </body>
</html>

Explanation :

  • app.set('view engine', 'ejs');: Sets EJS as the view engine for rendering HTML templates.
  • app.get('/', (req, res) => { ... });: Defines a route handler for the root URL. Inside the handler function, the res.render() method is used to render the 'index' template (assumed to be an EJS file) and pass data to it.


** Another Example of Connection to HTML**

Here's an example of passing employee details (name and age) from Express.js to an HTML template using EJS: 


const express = require('express');
const app = express();

// Set EJS as the view engine
app.set('view engine', 'ejs');

// Route handler for the root URL
app.get('/', (req, res) => {
   // Employee details
   const employee = {
       name: 'John Doe',
       age: 30
   };
   // Render the 'employee' template and pass employee details
   res.render('employee', { employee: employee });
});

// Start the server
app.listen(3000, () => {
   console.log('Server is running on port 3000');
});
       

<html>
   <head>
       <title>Employee Details</title>
   </head>
   <body>
       <h1>Employee Details</h1>
       <p>Name: <%= employee.name %></p>
       <p>Age: <%= employee.age %></p>
   </body>
</html>

Explanation :

  • const employee = { name: 'John Doe', age: 30 };: Defines an object containing employee details.
  • res.render('employee', { employee: employee });: Renders the 'employee' template (assumed to be an EJS file) and passes the employee object to it.

Custom Modules in Node.js

Here's an example of creating and using custom modules in Node.js: 


// mathUtils.js - Custom module
const add = (a, b) => {
   return a + b;
}

const subtract = (a, b) => {
   return a - b;
}

const multiply = (a, b) => {
   return a * b;
}

const divide = (a, b) => {
   if (b === 0) {
       throw new Error('Division by zero');
   }
   return a / b;
}

module.exports = { add, subtract, multiply, divide };
       

// app.js - Main file
const mathUtils = require('./mathUtils');

console.log(mathUtils.add(5, 3));        // Output: 8
console.log(mathUtils.subtract(10, 4));   // Output: 6
console.log(mathUtils.multiply(2, 6));    // Output: 12
console.log(mathUtils.divide(8, 2));      // Output: 4

Explanation :

  • mathUtils.js: This file defines a custom module with functions for basic math operations.
  • module.exports: Exports the functions add, subtract, multiply, and divide to be used in other files.
  • require('./mathUtils'): Imports the custom module mathUtils.js into the main file app.js.
