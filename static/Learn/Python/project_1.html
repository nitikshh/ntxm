Add Video To Drive By Python

We are Creating a Python Script to Just Add A Video to Google Drive 

import os
from googleapiclient.discovery import build
from googleapiclient.http import MediaFileUpload
from oauth2client.service_account import ServiceAccountCredentials
import progressbar

Explanation : import os: This imports the built-in os module, which provides a way of using operating system dependent functionality like reading or writing to the file system.

from googleapiclient.discovery import build: This imports the build function from the googleapiclient.discovery module, which is used to construct a Resource object for interacting with a Google API service.

from googleapiclient.http import MediaFileUpload: This imports the MediaFileUpload class from the googleapiclient.http module, which is used to upload files to Google services.

from oauth2client.service_account import ServiceAccountCredentials: This imports the ServiceAccountCredentials class from the oauth2client.service_account module, which is used for authenticating using a service account with OAuth 2.0.

import progressbar: This imports the progressbar module, which provides a visual indication of progress for long-running operations.

def get_file_size(file_path):

Explanation :

  • # Get the size of the file in bytes: This is a comment explaining that the function will return the size of the file in bytes.

Step 3

This script demonstrates how to authenticate using a service account and upload a video to Google Drive. It includes a function that handles the authentication process, uploads the video in chunks, and provides a progress bar to indicate the upload status. 

def authenticate_and_upload_video(video_path, folder_id):
    # Authenticate using service account credentials
    size = get_file_size(video_path)
    print(f"File size: {size:.2f} bytes")

    scope = [
        'https://spreadsheets.google.com/feeds',
        'https://www.googleapis.com/auth/drive'
    ]
    creds = ServiceAccountCredentials.from_json_keyfile_name(
        'client_secrets.json', scope)

    # Build Google Drive API service
    drive_service = build('drive', 'v3', credentials=creds)

    print('Uploading video to Google Drive............')

    # Set the chunk size to 1 MB
    chunk_size = 1024 * 1024
    media = MediaFileUpload(video_path, mimetype='video/mp4', resumable=True, chunksize=chunk_size)
    request = drive_service.files().create(
        body={
            'name': video_path.split('/')[-1],
            'parents': [folder_id]
        },
        media_body=media
    )

    bar = progressbar.ProgressBar(maxval=size, widgets=[progressbar.Bar('=', '[', ']'), ' ', progressbar.Percentage()])
    bar.start()

    response = None
    try:
        while response is None:
            status, response = request.next_chunk()
            if status:
                bar.update(min(status.resumable_progress, size))
    except Exception as e:
        print(f"Error during upload: {e}")
        print(f"Error details: {response}")
        bar.finish()
        return None

    bar.finish()
    print('Video upload completed.')

    video_id = response['id']  # Get the ID of the uploaded video
    print(f"Video uploaded successfully: {video_path.split('/')[-1]} (ID: {video_id})")

    return video_id

Explanation :

This function handles the authentication and upload process of a video to Google Drive:

  • def authenticate_and_upload_video(video_path, folder_id): Defines a function named authenticate_and_upload_video that takes two parameters: video_path (the path to the video file) and folder_id (the ID of the Google Drive folder).
  • size = get_file_size(video_path): Calls the get_file_size function to get the size of the video file in bytes and stores it in the variable size.
  • print(f"File size: {size:.2f} bytes"): Prints the size of the file to the console.
  • scope: Defines the scope for accessing Google Drive and Google Spreadsheets APIs.
  • creds = ServiceAccountCredentials.from_json_keyfile_name('client_secrets.json', scope): Authenticates using service account credentials from a JSON key file.
  • drive_service = build('drive', 'v3', credentials=creds): Builds the Google Drive API service using the authenticated credentials.
  • print('Uploading video to Google Drive............'): Prints a message indicating the start of the upload process.
  • chunk_size = 1024 * 1024: Sets the chunk size for the upload to 1 MB.
  • media = MediaFileUpload(video_path, mimetype='video/mp4', resumable=True, chunksize=chunk_size): Creates a MediaFileUpload object for the video file, specifying its MIME type and chunk size.
  • request = drive_service.files().create(body={'name': video_path.split('/')[-1], 'parents': [folder_id]}, media_body=media): Creates an upload request for the video file, setting its name and parent folder ID.
  • bar = progressbar.ProgressBar(maxval=size, widgets=[progressbar.Bar('=', '[', ']'), ' ', progressbar.Percentage()]): Initializes a progress bar to visually indicate the upload progress.
  • response = None: Initializes the response variable to None.
  • while response is None:: Starts a loop to upload the video in chunks until the upload is complete.
  • status, response = request.next_chunk(): Uploads the next chunk of the video file.
  • if status: bar.update(min(status.resumable_progress, size)): Updates the progress bar based on the number of bytes uploaded so far.
  • except Exception as e: ...: Handles any exceptions that occur during the upload process, prints the error message, and stops the progress bar.
  • video_id = response['id']: Retrieves the ID of the uploaded video from the response.
  • print(f"Video uploaded successfully: {video_path.split('/')[-1]} (ID: {video_id})"): Prints a success message with the video file name and its ID on Google Drive.
  • return video_id: Returns the ID of the uploaded video.

This script contains the main method that executes the video upload process. It specifies the video file path and the Google Drive folder ID, calls the function to authenticate and upload the video, and then prints the result. 

if __name__ == "__main__":
    video_path_to_upload = 'videos/How our solar system works.mp4'
    folder_id_to_upload = '1RUDPlOZH4SGM0JfF1LMwzTvnHaJlmRCj'

    uploaded_video_id = authenticate_and_upload_video(video_path_to_upload, folder_id_to_upload)

    if uploaded_video_id is not None:
        print(f"Uploaded video ID: {uploaded_video_id}")
    else:
        print("File upload failed.")

Explanation :

Here is the Main Method:

  • if __name__ == "__main__": This is a standard boilerplate in Python that ensures the following code block runs only if the script is executed directly, not when imported as a module.
  • video_path_to_upload = 'videos/How our solar system works.mp4': Defines the path to the video file that will be uploaded.
  • folder_id_to_upload = '1RUDPlOZH4SGM0JfF1LMwzTvnHaJlmRCj': Specifies the ID of the Google Drive folder where the video will be uploaded.
  • uploaded_video_id = authenticate_and_upload_video(video_path_to_upload, folder_id_to_upload): Calls the authenticate_and_upload_video function with the video path and folder ID as arguments, and stores the returned video ID in the variable uploaded_video_id.
  • if uploaded_video_id is not None: Checks if the upload was successful by verifying that uploaded_video_id is not None.
  • print(f"Uploaded video ID: {uploaded_video_id}"): If the upload was successful, prints the ID of the uploaded video.
  • else: print("File upload failed."): If the upload failed (i.e., uploaded_video_id is None), prints an error message.
